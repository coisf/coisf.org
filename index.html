致力于促进和发展中国的新一代开源基础软件

COISF 的使命是：促进中国下一代开源互联网基础软件生态系统的发展。

新一代的基础软件

Scale

面向云和更大规模的分布式系统设计，系统高度自治和自动化。

User Friendly

采用更现代的开发模式和部署方式，文档友好，业务开发者友好，监控友好。

Open

对贡献者友好，架构和选型上足够开放，尽量采用社区通用或事实标准组件，开发者能尽量无痛的参与进来增加新的特性。

Creative

面向未来的设计，用更先进的技术解决现代的问题。

旗下项目 (排名不分先后)


qihoo360/zippelin

Zeppelin 是为海量数据设计的半离线存储, 适合存储海量的图片, 视频小文件, 音频文件等等, 为线上业务直接提供服务. zeppelin 充分考虑半离线数据的特点, 数据总量大, 单value大, 对访问延迟要求不是特别苛刻, 资源成本等等. 目前正准备支持360内部的图床服务等等

baidu/bfs

BFS 是一个为实时应用设计的分布式文件系统，通过分布化的 NameServer 解决元数据单点问题，通过多机房的副本放置解决数据可靠性与可用性问题，除此之外，还针对读写延迟进行了特别的优化，当前支撑了包括百度网页库和用户行为分析在内的很多实时数据分析业务。

查看更多 »

baidu/tera

Tera 是一个高性能、可伸缩的结构化数据存储系统，被设计用来管理搜索引擎万亿量级的超链与网页信息。为实现数据的实时分析与高效访问，我们使用按行键、列名和时间戳全局排序的三维数据模型组织数据，使用多级 Cache 系统，充分利用新一代服务器硬件大内存、SSD 盘和万兆网卡的性能优势，做到模型灵活的同时，实现了高吞吐与水平扩展。

查看更多 »

pingcap/tidb

TiDB 是基于 Google F1 论文实现的分布式 NewSQL 数据库，定位是 100% 的 OLTP + 80%的 OLAP。 TiDB 是开源且 MySQL 兼容的，基本上就是一个可以无限水平扩展的 MySQL，提供分布式事务、跨节点 join、保证跨数据中心的数据的强一致性、故障自恢复的高可用、提供更快的查询和写入速度；对业务没有任何侵入性，简化开发，利于维护和平滑迁移。TiDB 与 TiKV 配合使用，作为 TiKV 的 SQL 层。

pingcap/tikv

TiKV 是基于 Google Spanner 论文实现的分布式 Key Value 数据库，支持弹性水平扩展以及高性能的跨行 ACID 分布式事务。
